Usando auto-py-to-exe (Alternativa):
Se preferir uma interface gráfica, você pode usar o auto-py-to-exe, que é uma interface para o PyInstaller. Instale-o com:

bash
Copy
pip install auto-py-to-exe
Depois, execute o comando:

bash
Copy
auto-py-to-exe
Isso abrirá uma interface gráfica onde você pode configurar as opções para gerar o executável.
